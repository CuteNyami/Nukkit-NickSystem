# Nukkit-NickSystem
Simple NickSystem for Nukkit server!

### - Features
* MySQL Support

### - Commands
* /nick
* /nick <nickname>
* /nick reset

### - HowToUse
You will find a file called mysql.json in plugins/NickSystem/ you need to file in you account data of MySQL 

then it will look like that!
```json
{
  "profile": {
    "host": "127.0.0.1",
    "database": "System",
    "user": "root",
    "password": "123456",
    "port": 3306
  }
}
```
